Feature: Sidebar links

  Scenario Outline: Sidebar redirect capabilites
    Given the standard_user is logged in
    And the sidebar is open
    When he presses the <Option>
    Then he should be redirected to <Url>

  Examples:
  | Option      | Url                                        |
  | 'All Items' | 'https://www.saucedemo.com/inventory.html' |
  | 'About'     | 'https://saucelabs.com/'                   |
  | 'Logout'    | 'https://www.saucedemo.com/'               |

  Scenario: Sidebar reset app state option
    Given the standard_user is logged in
    And the sidebar is open
    When he presses the 'Reset App State'
    Then the cart should be empty
    And the listed items should all have the 'Add to cart' button displayed
    And the filter used should change back to the 'Name (A to Z)'
    And the user should be redirected to the Inventory Page
